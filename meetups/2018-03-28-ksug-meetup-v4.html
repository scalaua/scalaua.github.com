Scala meetup V4 - Refactoring using Cats / CQRS with Akka

2018-03-28

Hey,

For our V4 meetup we'll tackle topics which can be useful for most of scala projects. We're lucky to have both speakers on-site this time.
First Vitalii Galchenko
(https://www.linkedin.com/in/vitalii-galchenko-aa3b395/) who is senior dev @ Quantel, will present "Refactoring using Cats higher level abstractions". During the talk you will learn some base abstractions of Cats library and how Monad transformers work. Also, we'll touch on how they can be used in Akka Http endpoints and Akka Http clients. Examples will include real project code where exception handling is substituted with Monad transformers.

For 2nd talk we will have Andrey Parhomenko (https://www.meetup.com/members/201449576/), who is a scala engineer working around Scala, Play and other frameworks. Andrey will explain "How to implement CQRS and event sourcing using Akka". In the talk you will get an overview of a new framework for building reactive microservice infrastructure which includes the concepts of reactive, resilient and immutable services along with CQRS and event sourcing.

Draft agenda:

- 18:30 - 18:45 - door opens - meet and greet
- 18:45 - 19:00 - talk prep
- 19:00 - 19:45 - Refactoring using Cats higher level abstractions (45 min) with Q/A included (in English)
- 19:45 - 20:00 - break.
- 20:00 - 20:45 - How to implement CQRS and event sourcing using Akka with Q/A included (talk lang TBA)
- 20:45 - beyond - pizza, refreshments and dev talk / discuss

Of course schedule is not set in stone I'm open to suggestions.

If you don't have any experience in Scala but still interested to learn feel free to join.

As before the we'll have a great location with night view of Kyiv, thanks to our new sponsors from INSCALE for sponsoring the meetup.
